// Nivel de Acceso

Tenemos 3 Enfoques de Hacking

Estos enfoques describen el nivel de información y acceso que tendremos al sistema que estamos evaluando.

Caja Blanca

Implica evaluar la seguridad de un sistema con el permiso del propietario, utilizando un enfoque completamente transparente.

Caja Gris

Combina técnicas de los enfoques de caja blanca y caja negra, permitiendo trabajar con acceso limitado a la información.

Caja Negra

Simulamos ataques reales y sofisticados sin tener información previa detallada sobre el sistema objetivo.
